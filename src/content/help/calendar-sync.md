# Calendar sync

Calendar sync makes booking pages and appointment workflows trustworthy. If your calendar is not accurate, the rest of the scheduling system gets shaky fast.

## What sync should cover

- Listing appointments
- Buyer consultations
- Personal blocked time
- Team calendars if shared coverage matters

## Before you turn it on

- Clean up duplicate calendars
- Decide which calendar is the source of truth
- Make sure time zones are correct

## Best practice

Use one main calendar for availability. Too many overlapping calendars usually create more confusion than value.
