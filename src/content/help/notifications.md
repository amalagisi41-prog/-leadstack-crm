# Notifications

Notifications only help when they guide action. Too many alerts make everything feel urgent and nothing feel clear.

## Good candidates for immediate alerts

- Hot new leads
- A showing request on an active listing
- A lead that replies after going quiet
- A failed send on an important follow-up

## Things that usually do not need instant interruption

- Routine activity logs
- Bulk import progress
- Low-priority status changes

## Simple rule

If an alert should change what you do in the next few minutes, send it. If not, let it live in the dashboard instead.
