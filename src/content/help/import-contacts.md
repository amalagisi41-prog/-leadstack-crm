# Import contacts

The fastest clean start is a CSV with one row per person. AgentStack can work with exports from Follow Up Boss, Chime, kvCORE, Real Geeks, Google Contacts, spreadsheets, or a hand-built list.

## What to include

- Name
- Email
- Phone
- Company or household name if you use one
- Tags or source notes if you already track them

## Before you import

1. Remove obvious junk rows.
2. Keep one primary email and one primary phone per contact where possible.
3. Make sure your column headers are readable.

## What happens in AgentStack

AgentStack shows a mapping preview before anything is written. You can confirm which column should become name, email, phone, tags, and source. If the same person already exists, the import flow shows likely duplicates before commit.

## Best practice

Start with your warm and active leads first. That gets your pipeline moving faster than waiting for a perfect full-database cleanup.
