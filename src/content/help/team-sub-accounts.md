# Team sub-accounts

Sub-accounts are how AgentStack keeps each agent organized without flattening everyone into one messy contact list.

## Why they matter

- Each agent keeps their own contacts and deals
- Brokers can still see the bigger picture
- Routing rules stay cleaner
- Permissions are easier to reason about

## Good uses

- Team leader with multiple agents
- Brokerage with territory-based routing
- ISA or admin support helping several producers

## Setup tip

Create the sub-account first, then invite the user into the right place. That keeps imports, lead routing, and AI settings from landing in the wrong workspace.
